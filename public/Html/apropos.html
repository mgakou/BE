Net-Simulate

A propos de Net-Simulate

Objectif du projet

Le projet Net-Simulate a été conçu dans le cadre d'un bureau d'études. C'est une application qui vise à simuler le routage IP à travers un ensemble de routes statiques.

Fonctionnalités

  • Gestion des réseaux: saisie, modification et suppression de réseaux et de sous-réseaux.
  • Gestion des hôtes: définition et manipulation des hôtes au sein des réseaux.
  • Routes statiques: création et gestion de routes statiques pour le routage des paquets.
  • Visualisation du réseau.
  • Simulation de routage.

Technologies utilisées

L'application s'appuie sur les technologies : HTML, CSS, JavaScript pour le front-end. Elle utilise PostGreSQL pour la gestion des bases de données côté serveur et PHP fait le lien entre les deux.
